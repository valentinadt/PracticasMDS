Para ver cÃ³mo afecta el caso del Diesel a ambos, es necesario preestablecer un periodo previo a la fecha y un periodo posterior, para poder trabajar con ellos. Estas fechas son las siguientes:

#fecha anterior al periodo
pre.period <- as.Date(c(start, "2015-09-18"))
#fecha posterior al periodo
post.period <- as.Date(c("2015-09-19", end))
impact_vw <- CausalImpact(vol, pre.period, post.period)
plot(impact_vw)

A continuaciÃ³n, se comprueba el efecto de BMW como regresor. Si se introduce el regresor, el efecto se aprecia aÃºn mÃ¡s acusante y la caida de los precios estÃ¡ claramente marcada y representada.

#estimar el grÃ¡fico con bmw como regresor
